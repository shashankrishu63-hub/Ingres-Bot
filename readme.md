# 🌊 INGRES-AI (Groundwater Data Assistant)  

**IngresBot** is a modern AI-powered web application that provides **detailed insights into India’s groundwater resources** using official **CGWB & IIT Hyderabad IN-GRES Website**.  
---

## 🚀 Features  

- **AI-Powered Chatbot** — Ask groundwater-related queries in any language (Hindi, English, Hinglish, etc.)  
- **Official Data Integration** — Powered by **INGRES (India Groundwater Resource Estimation System)**  
- **Detailed Explanations** — Statistics, rainfall data, aquifer details, categorization (Safe, Semi-Critical, Critical, Over-Exploited)  
- **Interactive Dashboard** — Sidebar navigation, stats cards, charts, and mapping  
- **Multi-Language Support** — Ask in any language → get response in the same language  
- **Modern UI/UX** — Responsive dark theme interface with animations  
- **Strict Domain** — Refuses non-groundwater queries (keeps focus on data)
